Prework Study Guide

HTML

  • The head element contains information about the webpage not visible to the user.
  • The body element represents the visible content shown to the user.
  • Header tages (H1, H2, H3, etc..) are used for headers, a page should only contain one H1 tag however.
  • Section elements are used to seperate different content or parts of a website.
  • P tags are used for text or paragraphs.
  • UL and OL tags are used for unordered and ordered lists, LI tags are used for list items.

CSS

  • The margin property sets the amount space around the outside of an element.
  • The padding property sets the amount of space around the content within an element.
  • The font-famlily property sets the font of an element.
  • The width and height properties set the width and the height of an element, this can be set as an absolute value or as a percentage.
  • The border property is used to create a border around an element, this can be customized with size, width and colour properties as well as many more.
  • The display property sets the rendering behaviour of en element, if set to block an element will start on a new line.

Git

  • git status: checks which brand we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git push origin branch-name: pushes changes to the current branch.
  • git pull origin main: pulls changes from the main branch and integrates them into your current branch.
  • git commit -m: commits changes to the GitHub repository with a commit message.
