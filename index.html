HTML

  • The head element contains information about the webpage not visible to the user.
  • The body element represents the visible content shown to the user.
  • Header tages (H1, H2, H3, etc..) are used for headers, a page should only contain one H1 tag however.
  • Section elements are used to seperate different content or parts of a website.
  • P tags are used for text or paragraphs.
  • UL and OL tags are used for unordered and ordered lists, LI tags are used for list items.
